Data Flow:
1. User creates asset → Frontend → SDK → Contract → On-chain storage
2. User buys fraction → Frontend → SDK → Contract → USDC transfer + Token mint
3. User transfers tokens → Frontend → SDK → Contract → SPL token transfer
4. Metadata retrieval → Frontend → SDK → Arweave/IPFS → Display

Key Features:
- Fractional ownership of real-world assets
- USDC-based trading
- Secure wallet integration
- Off-chain metadata storage
- Real-time portfolio tracking
- Multi-wallet support
